# 分治（Divide and Conquer）
分治是种解决问题的思想，在以往的题中有很多都用到分治的思想，比如二分查找、全排列，等等
### 分治步骤
1. 分解：将复杂问题分解为若干个小问题
2. 解决：用若子问题规模较小而容易被解决则直接解，否则递归地解各个子问题
3. 合并：将各个子问题的解合并为原问题的解
